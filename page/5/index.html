禁用Mac OS X Lion的Resume功能

08 Sep

　　Mac OS X Lion新增加的Resume功能在给使用者带来便利的同时也引发了隐私上的问题，没人希望尴尬的内容因此被呈现在他人眼前，可惜对于Resume，Lion并没有提供比较完善的配置功能，很多时候需要手动进行设置。

1. 关闭所有应用的Resume功能

　　System Preferences - General - Restore windows when quitting and re-opening apps 取消选中该选项。

2. 临时禁用某个应用的Resume功能

　　使用快捷键⌥⌘Q退出某个应用，下次再次打开时就不会恢复窗口，除非该应用关闭时正在编辑的文件尚未保存。

　　如果某个应用已经被退出，则可以在Finder前往~/Library/Saved Application State/文件夹，找到不想再次打开时恢复窗口的应用目录，如com.apple.Safari.savedState，将其删除即可（注：比较特别的是Preview在这个目录下只是一个链接，需要进入其目录后删除其下的所有文件）。

3. 永久关闭特定应用的Resume功能

　　以Safari为例，打开终端，输入以下命令：

defaults write com.apple.Safari NSQuitAlwaysKeepsWindows -int 0

　　要开启Resume功能，删除该键值即可：

defaults delete com.apple.Safari NSQuitAlwaysKeepsWindows

　　也可以使用Pref Setter、PlistEdit Pro等plist工具对应用的配置文件添加或删除NSQuitAlwaysKeepsWindows键值。

BTW: 不是使用Mac OS X Lion的API而是自己控制窗口恢复的应用使用上述方法无效，如Chrome 15.0.865.0 dev

关于iPhone4的那些事

08 Dec

　　11月3日在Apple Store上下单，13日拿到手，算算使用iPhone4也已经快一个月了，由于之前已经一直在使用Touch2，因此上手没有任何的问题，app、媒体库什么的都无缝过渡，这一个月摆弄之下也有不少感受。

　　外观：自被意外提前曝光起，我就一直觉得iPhone4的设计是iPhone系列中最具有苹果风格的了，玻璃、不锈钢元素的应用，线条刚硬之中又不失柔和，与Mac、MacBook Pro等苹果其它产品相得益彰，相较而言，前几代过分圆滑的线条以及塑料的外壳在其中反倒略显突兀，不是那么协调了。

　　Retina Display：毫无疑问，视网膜显示是iPhone4最重要的更新之一，而最终呈现的效果也确实让人惊艳，画面艳丽通透，尤其文本，在iBook上即使把字体调到最小，依然清晰锐利，相比之下，Touch2的颗粒明显，色彩泛白，略显混浊，小字体下更是惨不忍睹，使我不禁疑惑用iPhone4之前怎么会觉得Touch2的屏幕已经相当完美了呢，也开始当心以后别的屏幕还能看得下去吗？呵。

　　FaceTime：跟同事的Touch4尝试过一次，体验非常好，首先是连接上异常的便捷，基本不需要任何额外的操作，如拨打电话般快速，而使用中画面流畅，语音清晰，前后摄像头的自由切换更是一大亮点，如果今后在3G下也能够有类似表现的话，会是一个非常好的应用。

　　多任务处理：虽然并非真正意义上的多任务，但考虑到在移动设备上本来就负担过重的电池，这点取巧还是值得肯定的，切换过程也很炫，只是某些应用似乎对多任务支持还不是很好，切换着就崩溃退出了。

　　信号：平心而论，iPhone4信号表现确实无法让人百分百满意，我在公司班车上就多次遇到信号丢失的现象，这在我之前的索爱手机基本上是很少发生的，不过先前网络上盛传的“死亡之握”我倒一直没有遇到过，日常通话质量也相当的好，并未出现中途异常掉线等现象。

　　越狱：网上一直盛传一种说法，“没有越狱的iPhone只发挥了它50%的功能”，可能是我平常基本以网络应用为主，对这话并不太认可，对我而言“没有网络的iPhone只发挥了它50%的功能”才对，如果越狱只是为了多装两个游戏或改改运营商的图标，那对我来说真的就一点意义都没有，（限时）免费的游戏应用已经让我玩不过来了，实在需要的，偶尔再买一两个，已经足够，哪天我真越狱了，也是为了SBSettings的Toggles或能够单条删除通话记录。

　　数据网络：因为不想更换使用多年的手机号，我买的是无锁版，运营商选择的是中国移动，由于3G制式的原因，可以肯定的是iPhone在联通WCDMA网络下有更好的表现，但即使如我使用的是EDGE，体验也没有像网上某些人说的那么不堪，网络应用还是相当顺畅的，只是在打开某些较大页面时需要稍稍耐心等待罢了，像我们这些曾用56K猫爬过网的人来说，相比之下这样的等待已经很幸福了。呵

　　相机摄像：这方面我用的比较少，不过“500万、HDR、720p、LED闪光灯”这些参数对于像我这样从未考虑过要玩单反的傻瓜相机忠实用户来说，基本上应该够用了吧。

　　BTW：有些人对iOS的封闭很不耻，就我个人而言，相比Android版本分支极度混乱而言，我宁用选择支持iOS的封闭。

　　BTW2：以后考虑多写些跟iOS相关的东西，像app什么的。
