　　Mac下系统原生自带了全系统搜索工具Spotlight

http://sina.com/

快速启动应用

　　这是Alfred最基本的应用了，基本两个字母就能准确定位到想要的App了，勾选Fetures - Applications - Partail matching for Applications还能进行模糊匹配。

搜索文件

　　Alfred提供了诸如find open in等关键词用于搜索文件，不过个人感觉比系统自带的Spotlight还是稍逊一些，尤其是在Lion和Mountain Lion中均对Spotlight的功能做了大量的加强，所以如果是要搜索文件的话还是直接用Spotlight更加快速高效。

快速遍历文件系统

　　虽然Alfred的文件搜索功能稍弱，但遍历文件系统却是无比的方便， 在Alfred搜索框输入/（定位到系统根目录）或~（定位到当前用户的家目录）即可使用键盘快速遍历文件系统，遍历到所需文件后ENTER(若是文件夹则使用CMD+ O)可快速打开目标，CMD + ENTER可在Finder中定位目标，→可执行一些常用命令如拷贝、移动等。

　　快捷键CMD + OPTION + /或previous也可激活文件导航，同时它还会将文件路径定位到最近一次浏览的位置，CMD + OPTION + \或action则可激活当前Finder中选中项的常用命令操作。

控制iTunes

　　itunes或CMD + OPTION + ENTER可启动iTunes Mini Player，进行iTunes控制或媒体库搜索、遍历等操作，同时使用play next random等关键词命令也可对iTunes进行相应的快速控制。

自定义搜索

　　Alfred自带了大量网络搜索引擎，可在Features - Web Search中根据个人需要启用或禁用，当然还可以自定义搜索，如搜索淘宝http://s.taobao.com/search?q={query}，配合关键词tb就可以在Alfred的搜索框中直接搜索淘宝了。

访问剪切板

　　Alfred的剪切版 　　Alfred通过关键词clipboard或CMD + OPTION + C启动剪切版，

其它

  • 使用define关键词命令进行单词解释或spell拼写
  • 如同Spotlight，Alfred也支持直接对输入的数学表达式进行运算，或使用=关键词进行较为复杂的函数运算

一些小技巧

  • Advanced - Keyboard - Force Keyboard 选择U.S. 无论何时进入Alfred搜索框都将使用英文键盘，避免你正在使用中文输入法时进入Alfred搜索框需进行额外的输入法切换操作
  • 将Features - File Navigation - Advanced - Home Folder 从~ 改为`可以在快速进入当前用户家目录时少按一个按键SHEFT1
  • 如果在查找iTunes曲目时无法找到，可尝试通过Fetures - iTunes - Library - Reload Music Library重新加载媒体库2
