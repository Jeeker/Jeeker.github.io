FanfouPortable

03 Aug

　　饭否是我个人最喜欢的服务之一，如同其他的2.0网站，也有大量基于饭否API开发的工具可供选择使用，FanfouPortable就是一个简化版的饭否Wordpress插件，仅提供常用的新日志通告、消息发布、最新消息获取等基本功能，同时本插件对信息的发布与获取做了一定的优化，将对博客的执行效率的影响降到最低。

　　PS:在获取饭否上最新消息的过程中是不需要进行身份认证的，如果你只是想在你的博客上显示饭否上的最新消息则可不必在选项中设置密码。

  • 支持Widget
  • 新日志时发表通告饭否
  • 自动获取饭否上的最新消息
  • 简洁的信息发布
  • 向饭否发送或获取消息对站点访问的影响几乎可忽略不计

Usage

Install

解压，上传文件夹至wp-content\plugins目录下，激活，进入后台管理添加你的Fanfou ID和密码。

Function

　　FFP_GetPost($args='')
　　显示饭否上的最新消息，参数为字符串，支持的变量为：
　　limit 数量 输出数量 默认为10
　　noposttext 没有内容时输出的文字 默认None.
　　before 输出时每项前面的HTML代码 默认 <li>
　　after 输出时每项后面的HTML代码 默认 </li>
　　dateformat 日期格式 默认Y-m-d H:i:s
　　echo 是否直接输出到屏幕 默认 1
　　如<?php FFP_GetPost('limit=5&dateformat=H:i:s'); ?>，意即输出最新的5个饭否相信，且只显示时间，不显示日期

　　Note:ver 0.2后可通过Widget实现本函数功能。
